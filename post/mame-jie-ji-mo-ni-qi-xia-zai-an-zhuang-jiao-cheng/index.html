MAME街机模拟器下载安装教程

2025-10-13 3 min read # MAME # 街机模拟器

一、MAME模拟器介绍

MAME（全称 Multiple Arcade Machine Emulator，即 “多街机模拟器”）是一款专注于街机游戏模拟与arcade 硬件文化保护的开源软件，自 1997 年首次发布以来，已成为复古游戏领域的标杆工具，其核心价值不仅是 “玩老游戏”，更在于对即将消失的街机硬件与游戏历史的数字化存档。

MAME街机模拟器主要有几下特点：

  • 支持的街机游戏最多；
  • 官方文档详细、社区活跃度高；
  • 跨平台（兼容Windows、MacOS、Linux、Android、IOS）；
  • 支持金手指、即时存档、着色器、装饰图等众多功能；

二、MAME模拟器下载安装

1、下载安装模拟器

  • 下载页面：https://www.mamedev.org/release.html
  • 进入MAME官网下载页面，选择最新版本下载，执行exe文件后可选择目录进行安装。

2、模拟器UI界面介绍

双击mame.exe进入模拟器：

  • 页首：页首这里可以直接键盘输入信息搜索游戏。页首下方有四个按钮分别是收藏/移除喜爱的游戏，导出当前游戏列表信息至文件，校验游戏和显示dat文件。
  • 左侧边栏：根据游戏的不同属性默认创建的分类目录，自己也可以创建分类，后续再详细讲讲。
  • 游戏列表：中间为游戏列表，双击游戏可开始游玩。
  • 右侧边栏：右侧主要展示周边文件，如游戏海报图、预览图等各种元数据信息，这些数据模拟器没有自带，需要自行下载放入相应的文件夹中。
  • 设置菜单：General Settings为全局设置，System Setings为当前游戏设置（先在游戏列表中选中游戏后，再点此进行设置）。
  • 页脚：游戏状态信息。
    • 绿色：通常表示游戏可以完美运行，模拟器对该游戏的支持较好，理论上不会存在明显的兼容性或运行问题。
    • 黄色：意味着游戏或多或少存在一些问题，可能是画面显示异常、音效缺失或部分功能无法正常使用等，但一般仍可运行。
    • 红色：表示游戏不能正常运行，可能是由于缺少必要的 ROM 文件、模拟器版本不兼容或其他严重问题，导致游戏无法启动或运行时会出现严重错误。

3、设置UI页面语言为中文

MAME模拟器带有简体中文语言包，翻译不是特别完美，但是不影响使用。

中间的游戏列表部分需要创建一个mame_cn.lst放在模拟器根目录，文件编码为UTF-8。然后自行在里面每行写入游戏压缩包名、按Tab键分割、再写入中文游戏名称。目前已经有大佬整理了这个文件，可联系我分享给你们，就不需要自己花时间翻译了。

准备上面的步骤后，依次点击General Settings，Custommize UI，将Language、System Names按图示设置，设置好后按Esc键返回，记得点击保存设置。
